Web Development

Website Sistem Inventori

Merancang sistem inventori berbasis web menggunakan framework Laravel dengan fitur CRUD (Create, Read, Update, Delete). Sistem ini menyajikan informasi lengkap terkait barang, kategori, dan stok, serta mengelola pencatatan transaksi barang masuk dan keluar secara efektif. Selain itu, dilengkapi fitur autentikasi (login dan registrasi) untuk memastikan akses yang aman dan terstruktur bagi pengguna.

Internet Of Things (IOT)

Monitoring Suhu & Kelembaban

Merancang simulasi sistem monitoring suhu dan kelembaban berbasis Internet of Things (IoT) menggunakan ESP32 dan sensor DHT22 melalui platform Wokwi. Data yang diperoleh akan dikirim secara real-time ke ThingsBoard dan divisualisasikan dalam bentuk dashboard interaktif berupa gauge untuk memantau nilai temperatur dan humidity secara langsung.

Internet Of Things (IOT)

RC-Car Wifi

Merancang RC-Car berbasis WiFi menggunakan NodeMCU ESP8266 sebagai project tim. Kendaraan ini dapat dikendalikan secara real-time melalui aplikasi “ESP8266 WiFi Robot Car”, dengan proses pemrograman dilakukan melalui Arduino IDE. Untuk meningkatkan kemudahan penggunaan, proyek ini juga dilengkapi dengan dashboard kontrol sederhana berbasis MIT App Inventor sehingga pengoperasian RC-Car menjadi user-friendly.

Internet Of Things (IOT)

Air Purifier

Merancang Air Purifier sebagai salah satu project tim berbasis IoT untuk meningkatkan kualitas udara dalam ruangan tertutup dengan mengurangi gas berbahaya. Sistem ini menggunakan sensor MQ-2 (hidrogen dan metana), MQ-7 (karbon monoksida), dan MQ-135 (amonia dan karbon dioksida) untuk mendeteksi gas di udara. Udara yang terdeteksi tercemar akan dihisap masuk dan disaring melalui tiga lapisan filter penyaring, lalu dikeluarkan kembali melalui bagian belakang.

Infrastructure as a Service (IaaS)

Membangun Jaringan Internet Perpustakaan

Merancang simulasi jaringan internet perpustakaan dua lantai menggunakan MikroTik di GNS3 dengan topologi terpusat yang menghubungkan ruang baca dan laboratorium komputer. Simulasi ini dikembangkan sebagai project tim yang mencakup konfigurasi routing, IP addressing, distribusi melalui switch, serta dukungan koneksi nirkabel untuk menciptakan jaringan yang efisien dan scalable.
